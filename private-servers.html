üîß Troubleshooting

Common Issues and Solutions

üéÆ Console Private Server Issues

The developers are aware of issues with console private servers. If you're experiencing problems on Xbox or PlayStation, try accessing through PC/mobile or wait for the official fix.

Can't Join Private Server

Check if you have permission to join, ensure the server is still active, and verify your internet connection. The server owner may need to re-invite you.

Server Not Loading

Try refreshing the page, clearing your browser cache, or restarting the Roblox client. Server maintenance might also cause temporary issues.

Payment Issues

Ensure you have sufficient Robux and check your payment method. Contact Roblox support if payment problems persist.
