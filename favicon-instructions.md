## To Create favicon.ico File

### Option 1: Online Generator
1. Go to https://favicon.io/favicon-generator/
2. Use these settings:
   - Text: ⚒️ (hammer emoji)
   - Background: Rounded, #1a1a1a (dark)
   - Font Family: Any readable font
   - Font Size: Large
3. Download the generated favicon.ico
4. Place it in the root directory as `/favicon.ico`

### Option 2: From Existing Design
1. Create a 32x32 pixel image with the hammer emoji
2. Use https://convertio.co/png-ico/ to convert to ICO format
3. Save as favicon.ico in root directory

### Option 3: Use Existing SVG
The current inline SVG favicon should work well for most browsers and search engines, including Yandex.
