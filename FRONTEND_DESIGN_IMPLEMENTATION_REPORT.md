### 1. **CSS 变量系统**
```css
:root {
    /* 字体系统 */

    --font-gaming: 'Press Start 2P', monospace;
    --font-body: 'Inter', 'Space Grotesk', sans-serif;
    --font-code: 'JetBrains Mono', 'Fira Code', monospace;
    
    /* 阴影系统 */
    --shadow-forge-glow: 0 0 20px rgba(245, 158, 11, 0.3);
    --shadow-card-hover: 0 8px 32px rgba(0, 0, 0, 0.4);
    
    /* 动画时间 */
```
